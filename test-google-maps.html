🗺️ Google Maps API Key 测试

🔧 如何获取 API Key:

  1. 访问 Google Cloud Console
  2. 创建或选择项目
  3. 启用 "Maps JavaScript API"
  4. 创建凭据 → API 密钥
  5. 在 API 密钥限制中添加以下域名：
    • https://market-link-express.com/*
    • https://www.market-link-express.com/*
    • https://market-link-express.netlify.app/*
    • https://localhost/* (用于本地测试)
