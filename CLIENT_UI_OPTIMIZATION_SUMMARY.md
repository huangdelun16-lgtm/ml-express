## 🎨 优化原则

### 字体对齐
1. **文本内容**：统一左对齐（中文、英文、缅甸语）
2. **表单输入**：统一左对齐
3. **按钮文本**：统一居中对齐
4. **标题**：根据上下文选择左对齐或居中对齐

### UI 设计
1. **统一间距**：使用 CSS 变量 `--spacing-*`
2. **统一字体大小**：使用 CSS 变量 `--font-size-*`
3. **统一圆角**：使用 CSS 变量 `--radius-*`
4. **统一阴影**：使用 CSS 变量 `--shadow-*`
5. **统一颜色**：使用 CSS 变量 `--color-*`

### 响应式设计
1. **移动端** (< 768px)：较小的字体和间距
2. **平板** (768px - 1023px)：中等字体和间距
3. **桌面端** (≥ 1024px)：较大的字体和间距
