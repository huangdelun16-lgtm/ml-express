## 🛠️ 故障排除

### 构建失败
1. 检查 `package.json` 中的构建脚本
2. 检查环境变量是否正确设置
3. 查看构建日志中的错误信息
4. 确保所有依赖都已安装

### 运行时错误
1. 检查环境变量是否正确
2. 检查 Google Maps API Key 是否有效
3. 检查 Supabase 连接是否正常
4. 查看浏览器控制台错误

### 性能问题
1. 检查 Vercel Analytics 中的性能报告
2. 优化图片和静态资源
3. 使用 Vercel 的缓存策略
4. 考虑使用 Vercel Edge Functions
