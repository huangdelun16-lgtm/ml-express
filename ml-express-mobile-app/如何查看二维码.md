# 📱 如何查看 Expo 二维码

## 方法 1：在终端中查看（最简单）

1. 打开终端
2. 进入项目目录：
   ```bash
   cd /Users/aungmyatthu/Desktop/ml-express/ml-express-mobile-app
   ```
3. 启动开发服务器：
   ```bash
   npm start
   ```
4. 终端会显示一个二维码，类似这样：
   ```
   ██████████████████████████████████████
   ██████████████████████████████████████
   ██████████████████████████████████████
   ...
   ```
5. 用手机上的 **Expo Go** App 扫描这个二维码

## 方法 2：在浏览器中查看

1. 启动服务器后，终端会显示一个 URL，例如：
   ```
   Metro waiting on exp://192.168.1.100:8081
   ```
2. 或者访问：http://localhost:8081
3. 浏览器中会显示二维码和连接信息

## 方法 3：使用 Tunnel 模式（推荐，适合远程连接）

```bash
cd /Users/aungmyatthu/Desktop/ml-express/ml-express-mobile-app
npx expo start --tunnel
```

这会创建一个公共 URL，可以在任何地方访问。

## 📲 使用 Expo Go 扫描

1. **Android**: 在 Google Play 搜索 "Expo Go" 并安装
2. **iOS**: 在 App Store 搜索 "Expo Go" 并安装
3. 打开 Expo Go App
4. 点击 "Scan QR Code"（扫描二维码）
5. 扫描终端或浏览器中显示的二维码
6. 应用会自动加载！

## 🔗 快速启动命令

```bash
# 进入项目目录
cd /Users/aungmyatthu/Desktop/ml-express/ml-express-mobile-app

# 启动开发服务器（显示二维码）
npm start

# 或者使用 tunnel 模式（公共链接）
npx expo start --tunnel
```

## 💡 提示

- 确保手机和电脑在**同一个 WiFi 网络**中
- 如果使用 tunnel 模式，不需要在同一网络
- 二维码会在终端中持续显示，随时可以扫描
