# Google Play Feature Graphic 设计规范

## 📐 基础规格

### 尺寸要求
- **宽度**: 1,024 px
- **高度**: 500 px
- **宽高比**: 2.048:1 (横向横幅)
- **分辨率**: 72 DPI 或更高
- **格式**: PNG 或 JPEG
- **文件大小**: 最大 15 MB
- **颜色模式**: RGB

## 🎯 设计元素说明

### 1. Logo 使用
- **位置**: 左上角或左侧中心
- **尺寸**: 80-100px (正方形)
- **背景**: 透明或白色圆形背景
- **文件**: 使用 `ml-express-mobile-app/assets/logo.png`

### 2. 功能图标
建议使用以下图标（可用图标库如 Feather Icons, Material Icons）：
- 📦 **包裹图标** - 代表"我的任务"
- 🗺️ **地图图标** - 代表"配送地图"
- 📷 **相机图标** - 代表"扫码查询"
- 📊 **统计图标** - 代表"个人中心"

**图标规格**:
- 尺寸: 64-80px
- 颜色: #FFFFFF 或品牌蓝色
- 样式: 线性图标或填充图标
- 排列: 横向排列，间距均匀

### 3. 背景设计
**推荐方案1: 渐变背景**
```
使用品牌蓝色渐变:
- 左上角: #b0d3e8
- 右下角: #1B5E7A
- 渐变方向: 135度 (左上到右下)
```

**推荐方案2: 纯色背景 + 装饰**
```
- 主背景: #2E86AB
- 添加几何图形装饰（圆形、线条）
- 使用低透明度白色叠加层
```

**推荐方案3: 抽象图案**
```
- 使用快递/配送相关的抽象图案
- 地图路线、包裹轮廓等
- 低透明度，不干扰文字
```

## ✨ 设计细节建议

### 1. 视觉层次
- **第一层**: Logo + 应用名称（最突出）
- **第二层**: 副标题/标语
- **第三层**: 功能图标（辅助说明）

### 2. 对比度
- 确保文字在背景上清晰可读
- 使用文字阴影或描边增强可读性（如需要）
- 文字阴影: `0 2px 4px rgba(0, 0, 0, 0.3)`

### 3. 装饰元素
- 可以添加微妙的几何图形
- 使用品牌色的半透明圆形或线条
- 避免过度装饰，保持简洁

### 4. 响应式考虑
- 虽然尺寸固定，但要确保在小屏幕上查看时仍然清晰
- 重要元素不要放在边缘（可能被裁剪）

## 🎨 Figma 设计步骤

### Step 1: 创建画布
1. 打开 Figma
2. 创建新 Frame
3. 设置尺寸: **1024 × 500**
4. 命名: "Feature Graphic - ML Express Staff"

### Step 2: 设置背景
1. 创建矩形覆盖整个画布
2. 应用渐变填充:
   - Type: Linear
   - Angle: 135°
   - 颜色1: #b0d3e8 (位置 0%)
   - 颜色2: #1B5E7A (位置 100%)

### Step 3: 添加 Logo
1. 导入 `logo.png`
2. 调整尺寸至 80-100px
3. 位置: 左侧，距离左边 60px，垂直居中或偏上
4. 可选: 添加白色圆形背景（半径 50px，填充白色，透明度 90%）

### Step 4: 添加应用名称
1. 创建文本层
2. 文字: "ML Express Staff"
3. 字体: Inter Bold 或 Roboto Bold
4. 字号: 52px
5. 颜色: #FFFFFF
6. 位置: Logo 右侧，垂直对齐
7. 添加文字阴影: `0 2px 8px rgba(0, 0, 0, 0.3)`

### Step 5: 添加副标题
1. 创建文本层
2. 文字: "Professional Courier Delivery Management"
3. 字体: Inter Medium 或 Roboto Medium
4. 字号: 22px
5. 颜色: rgba(255, 255, 255, 0.95)
6. 位置: 主标题下方 16px

### Step 6: 添加功能图标
1. 在右侧区域添加图标组
2. 使用图标库（Feather Icons 或 Material Icons）
3. 图标尺寸: 72px
4. 图标颜色: #FFFFFF
5. 图标间距: 28px
6. 排列: 水平排列，垂直居中

### Step 7: 添加装饰元素（可选）
1. 添加半透明圆形（半径 150px，填充 #FFFFFF，透明度 10%）
2. 位置: 右上角或右下角
3. 可以添加多个，大小不一，创造层次感

### Step 8: 最终检查
- [ ] 尺寸: 1024 × 500
- [ ] 所有文字清晰可读
- [ ] Logo 清晰可见
- [ ] 颜色符合品牌规范
- [ ] 元素对齐整齐
- [ ] 没有重要元素在边缘

## 📤 导出设置

### Figma 导出步骤
1. 选择整个 Frame
2. 右侧面板 → Export
3. 格式选择: **PNG** 或 **JPEG**
4. 分辨率: **1x** (1024 × 500)
5. 质量: **100%** (JPEG) 或 **PNG-24** (PNG)
6. 导出

### 文件命名
```
feature-graphic-ml-express-staff.png
或
feature-graphic-ml-express-staff.jpg
```

### 文件大小检查
- 确保文件大小 < 15 MB
- 如果过大，可以适当降低 JPEG 质量（但不要低于 85%）

## 📝 额外提示

1. **保持简洁**: Feature graphic 会在小屏幕上显示，不要添加太多细节
2. **品牌一致性**: 确保与应用的视觉风格一致
3. **测试预览**: 在 Google Play Console 中上传后，在不同设备上预览效果
4. **备用方案**: 可以设计 2-3 个版本，选择最佳效果
