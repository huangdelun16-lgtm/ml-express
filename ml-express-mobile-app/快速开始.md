## 🚀 现在就能用！（3步骤）

### Step 1: 在手机上安装 Expo Go

打开手机（Android 或 iPhone）：
1. 打开应用商店
2. 搜索 **"Expo Go"**
3. 下载安装

### Step 2: 查看终端

在你的 Mac 终端中，应该能看到：
- 一个二维码
- 或一个网址（如：exp://192.168.x.x:8081）

**如果没看到**，运行：
```bash
cd /Users/aungmyatthu/Desktop/MLEXPRESS/MLExpressApp
npm start
```

### Step 3: 扫码打开应用

**Android 手机**：
- 打开 Expo Go
- 点击 "Scan QR Code"
- 扫描终端中的二维码

**iPhone**：
- 打开相机
- 扫描二维码
- 点击打开链接

**等待10-30秒** → 应用就会出现在手机上！🎉

## 📦 构建真正的 App（可选）

### Android APK
```bash
# 安装 EAS CLI
npm install -g eas-cli

# 登录 Expo 账号（免费注册）
eas login

# 构建 Android APK
eas build --platform android --profile preview
```

构建完成后，下载 APK 文件，安装到手机！

### iOS（需要 Apple Developer 账号）
```bash
eas build --platform ios
```
