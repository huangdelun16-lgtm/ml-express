## 🔧 构建和上传

### 6. 构建应用 ⚠️
- [ ] **安装 EAS CLI**: `npm install -g eas-cli`
- [ ] **登录 Expo**: `eas login`
- [ ] **配置凭据**: `eas credentials`（首次）
- [ ] **构建 App Bundle**: `eas build --platform android --profile production`
- [ ] **下载 .aab 文件**: 从 expo.dev 下载

### 7. Google Play Console ⚠️
- [ ] **创建应用**: 在 Google Play Console 创建新应用
- [ ] **填写基本信息**: 名称、描述、分类
- [ ] **上传图标**: 512x512 像素
- [ ] **上传截图**: 至少 2 张
- [ ] **设置隐私政策 URL**: `https://mlexpress.com/privacy`
- [ ] **完成内容分级**: 填写分级问卷
- [ ] **上传 App Bundle**: 上传 .aab 文件
- [ ] **填写版本说明**: 描述新功能和改进

### 8. 提交审核 ⚠️
- [ ] **预览商店页面**: 检查所有信息
- [ ] **验证所有链接**: 确保可访问
- [ ] **提交审核**: 点击"提交以供审核"
- [ ] **等待审核**: 通常 1-3 个工作日
