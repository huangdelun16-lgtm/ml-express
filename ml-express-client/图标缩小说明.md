## 🔄 如何查看更改效果

### 方法 1：清除缓存并重新启动（推荐）

1. **停止当前的 Expo 服务器**（在终端按 `Ctrl+C`）

2. **清除缓存并重新启动**：
   ```bash
   cd ml-express-client
   npx expo start --clear
   ```

3. **在 Android 设备上重新加载应用**：
   - 在 Expo Go 中，摇动设备打开开发者菜单
   - 选择 "Reload"（重新加载）
   - 或者完全关闭并重新打开 Expo Go 应用

### 方法 2：完全重新构建（如果方法 1 不起作用）

1. **清除所有缓存**：
   ```bash
   cd ml-express-client
   rm -rf node_modules/.cache
   npx expo start --clear
   ```

2. **在 Android 设备上**：
   - 完全关闭 Expo Go 应用
   - 重新打开 Expo Go
   - 重新扫描二维码或输入 URL

### 方法 3：检查 Android 系统字体大小设置

Android 系统字体大小设置可能会影响 emoji 图标的显示大小：

1. 打开 Android 设备的 **设置**
2. 找到 **显示** 或 **字体大小**
3. 将字体大小设置为 **标准** 或 **小**
4. 重新加载应用

- 如果图标仍然没有缩小，请确保：

## 🐛 故障排除

如果图标仍然没有缩小，请尝试：

1. **完全卸载并重新安装 Expo Go**：
   - 卸载 Expo Go
   - 重新从 Google Play Store 安装
   - 重新扫描二维码

2. **使用开发构建**（如果可用）：
   ```bash
   cd ml-express-client
   eas build --profile development --platform android
   ```

3. **检查代码是否正确应用**：
   - 确认所有文件的修改都已保存
   - 确认 Git 提交包含所有更改
   - 检查是否有其他样式覆盖了图标大小
