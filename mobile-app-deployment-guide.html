🔧 客户端应用部署 (ml-express-client)

用于客户下单和跟踪包裹

  1. 进入客户端应用目录
    cd ml-express-client
  2. 安装依赖
    npm install
  3. 构建应用
    npx expo build:android
    npx expo build:ios
  4. 或者使用 EAS Build (推荐)
    npx eas build --platform android
    npx eas build --platform ios

🔧 Staff 应用部署 (ml-express-mobile-app)

用于骑手接单和配送

  1. 进入 Staff 应用目录
    cd ml-express-mobile-app
  2. 安装依赖
    npm install
  3. 构建应用
    npx expo build:android
    npx expo build:ios
  4. 或者使用 EAS Build (推荐)
    npx eas build --platform android
    npx eas build --platform ios

⏰ 重要提醒

Google Cloud Console 配置：

  • 确保 API Key 已启用以下服务：
    • Maps SDK for Android
    • Maps SDK for iOS
    • Maps JavaScript API
  • 确保域名限制包含：
    • Android Package: com.mlexpress.client
    • iOS Bundle ID: com.mlexpress.client
    • Android Package: com.marketlinkexpress.staff
    • iOS Bundle ID: com.marketlinkexpress.staff
